# local/

Scratch space for files that should **not** be committed: experiments, draft artifacts, scratch notes, local secrets, ad-hoc scripts, and throwaway data.

Everything in this folder is ignored by Git **except this `README.md`** (so the folder is kept in the repo). See the rule in [`.gitignore`](../.gitignore).

## Use it for
- Drafting a prompt/agent before it's ready to move into its real folder.
- Local `.env` files and credentials.
- Eval run outputs and temporary datasets.
- Anything personal you don't want to share.

> ⚠️ Nothing here is versioned or backed up. Move artifacts to their proper folder once they're ready.
