Image vs. container?
Image = immutable template; container = running instance of it.
What is a registry?
A store of images to pull/push (e.g. Docker Hub).
-p 8080:80 means?
Host port 8080 → container port 80.
Persist container data?
Mount a volume: -v name:/path.
Risk of :latest?
It's just the default tag, not "newest stable" → pin a version.
Build an image?
docker build -t name . (from a Dockerfile).
Why are containers light?
They share the host kernel; no full guest OS like a VM.
Bind address inside a container?
0.0.0.0 (so the host can reach it), not 127.0.0.1.
