Mastery rubric

You've reached the beginner bar when you can do all of the following.

  • Explain, without rote, what a container is and the problem it solves.
  • State the difference between an image and a container, and give an example of each.
  • Run any public image and publish a port, from memory.
  • Write a basic Dockerfile and build an image from it.
  • Explain why container data is ephemeral and how a volume fixes it.
  • Identify at least three common mistakes and their fixes.
  • Debug a crashing container using docker logs / exec.
  • Say honestly when Docker is not the right tool, and what this capsule did not cover.
